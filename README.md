# React Mui DashBoard

提供后台管理中常用的基础功能和基础页面方便快速开发

## 目录结构
```

├── src
│   └── containers         Page每个页面的最外层容器
        └── Dashboard.tsx  仪表板
        └── errorpage      错误页面
        └── Root.tsx       根
│   └── index.ts
└── tsconfig.json
├── package.json            
├── README.md              项目说明文件
```
